UI Frameworks

The students understand the basics and concepts of the development of modern graphical user interfaces, in theory and using modern architectures and frameworks.They have used frameworks to develop to develop examples.The students are able to read research papers and can use this knowledge in their own projects.

Content

  • User interfaces from 1970 to now
  • Basic concepts of user interfaces
  • Requirements and standards
  • Technical aspects
    • Technologies
    • Interaction techniques
  • Architecture and design for graphical interactive systems
  • Declarative software systems
  • Frameworks and toolkits for user interfaces
  • Interface testing
