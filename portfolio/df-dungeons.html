  • The Azure Vault
  • Algeth'ar Academy
  • Halls of Infusion
  • Brackenhide Hollow
  • Neltharus
  • Uldaman (legacy of tyr)
Check out Clean Icons Mechagnome Edition if you want HD icon textures. If you like to change the Size of the Icons, please use the "Group Scale" option instead of changing width/height of all Auras at the same time because that may crash your game while Group-Scale is lag-free. If you want to have timers and are not using OmiCC/Elvui select the group , go to display options and tick "show text" below text1. If you want to have some sounds, use this addon: SharedMedia Causese If you are using Causeses addon and do NOT want sounds select the group and go to Actions -> on Show and untick play sound Every feedback/help is greatly appreciated feel free to contact me by joining my .
